# 使用方法

## 环境要求
- Python 3.7+
- pip

## Windows用户
1. 双击运行 `start.bat`
2. 在菜单中选择：
   - `0`: 运行配置向导
   - `1`: 直接启动代理服务器

## Linux/macOS用户
1. 赋予启动脚本执行权限：
   ```bash
   chmod +x start.sh
   ```
2. 运行启动脚本：
   ```bash
   ./start.sh
   ```
3. 在菜单中选择：
   - `0`: 运行配置向导
   - `1`: 直接启动代理服务器

## 配置说明
1. 首次使用请选择 `0` 进行配置
2. 配置完成后可选择是否直接启动代理
3. 配置文件保存在 `config.json`

## 代理服务器
- 默认地址：`http://127.0.0.1:9876/`
- API格式：与OpenAI API兼容
- 支持流式输出
